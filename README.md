# Proyecto final de programación web
En el presente repositorio se guardará el proyecto del curso "Taller de Programación Web", el cual aplicaremos conocimientos de JavaScript, HTML y CSS para la creación de una red social básica, la cual contará con un feed donde se verán diversas publicaciones y sus respectivos comentarios.

## Objetivo principal
Buscar la interacción, conexión e intercambio de información de los usuarios finales de una manera sencilla, demostrando que la creación de dicho entorno es posible si adquirimos los conocimientos necesarios.
